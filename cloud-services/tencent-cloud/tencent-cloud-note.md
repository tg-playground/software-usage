# Note of Tencent Cloud 

### 腾讯云服务器

#### 购买云服务器

购买实例

- 包年包月/按量计费
- 存储
- 网络宽带。按带宽计费/按流量计费

购买后，几分钟之后才能在云控制台看到自己购买的实例。

实例硬件配置

标准型S2

- 1 核 2 GB 1 Mbps
- 系统盘：50GB，高性能云硬盘

#### 配置

设置实例安全组

设置实例SSH密钥

#### 操作

使用 XShell + SSH 密钥登录云服务器
